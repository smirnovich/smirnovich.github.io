## What is a VHDL Package

In addition to a testbench-files and regular synthesis-files VHDL provides library-files called **packages**. Package is a usefull file to collect some universal data so it would not increase the size of the design files and can be easily shared from developer to developer. This kind of files is desribed by VHDL LRM and must be supported with all vendors. For now on, I've checked it with:

* Quartus 18.0
* Modelsim 10.5
* Vivado/Vsim 2019.2
* All edaplayground.com instruments

## Synthax
The ``ieee`` library  also consist of packages:  ``numeric_std``, ``std_logic_1164`` are also packages.

The basic content of package is shown below:

```vhdl
--my_pkg.vhd:

package my_pkg is
    constant EXAMPLE: integer := 123;
end package my_pkg;

package body my_pkg is

end package body my_pkg;
```
